# ⛏️ Minecraft English

Красочная интерактивная игра в стиле Minecraft для изучения английских слов на уроке (для детей-начинающих). Все слова общие (cow, axe, tree, apple...), сгруппированы по категориям: Animals, Nature, Items, Food.

## Режимы

| Режим | Что делает |
|---|---|
| 🃏 **Learn** | Карточки: картинка → кнопка **REVEAL** открывает и озвучивает слово → ребёнок отмечает ✅ «знаю» / ❌ «ещё учу». В конце можно повторить только сложные слова. |
| 🔤 **Spell It** | Собери слово из букв — drag & drop (или просто тапай по буквам). |
| ❓ **Quiz** | Картинка + 4 слова на выбор. |
| 🧠 **Memory** | Найди пары «картинка — слово». |
| 🔊 **Sound Hunt** | Мини-игра на время: слушай слово и успей нажать нужную картинку. |

Везде есть кнопка **◀ BACK**, кнопка полноэкранного режима **⛶**, озвучка слов (встроенный синтез речи браузера), звуковые эффекты и конфетти. Прогресс «выученных» слов сохраняется в браузере.

## Запуск

```bash
npm install     # один раз
npm run dev     # запустить для разработки (http://localhost:5173)
npm run build   # собрать готовую версию в папку dist/
```

Собранную папку `dist/` можно открыть на любом компьютере (лучше через любой локальный сервер, но благодаря `base: './'` работает и просто открытием `index.html`).

## Свои картинки

Кладите PNG-файлы в `public/assets/images/` с именем слова: `cow.png`, `axe.png`...
Нет картинки — игра покажет эмодзи, всё продолжит работать.

## Добавить/изменить слова

Список слов — в файле `src/data/words.js`. Формат:

```js
{ word: 'cow', emoji: '🐄', cat: 'animals' }
```

Категории: `animals`, `nature`, `items`, `food`.

## Технологии

React 18 + Vite. Озвучка — Web Speech API, звуки — Web Audio API (без аудиофайлов), прогресс — localStorage. Интернет нужен только для пиксельного шрифта (без него включится запасной).
